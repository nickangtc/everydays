Genuary day 10 prompt: Generative music.

What are the basics of generative music? I think it's loading a file and then manipulating the playback rate, volume, and some other musical detail like that with some randomness.

Since I'm so new to this, I'd rather not write randomness into the program but rather expose sliders so I can hear how difference the sound is when it's adjust a certain way.
