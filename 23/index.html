Glitch art (debug view)

Notes

Following Genuary 2023 prompt for day 5: "Debug view". For this project I'm reusing code from Glitch art made 2 days ago and focusing on adding a debug view that can be seen and used from the browser.

My intent with today's project is to write and expose some controls on the browser UI that will let me make on-the-fly adjustments to variables that would change the output image on the canvas, a rudimentary implementation similar to what controlP5 enables for the Processing environment.

Created with p5.js, a JavaScript library for creative coding.
